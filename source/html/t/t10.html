Планирование среды Life Touch Installation

Краткое руководство по началу работы с установкой Lite Touch

Microsoft Deployment Toolkit (MDT) 2013 предоставляет технологию для развертывания операционных систем Windows и Microsoft Office. Это руководство поможет вам быстро оценить MDT 2013, предоставив сжатые пошаговые инструкции по ее использованию для установки операционной системы Windows 8.1 с помощью Lite Touch Installation (LTI) с помощью загрузочного носителя (DVD-диска или USB-устройства флэш-памяти).

После использования этого руководства для оценки MDT ознакомьтесь с остальными рекомендациями по MDT, чтобы узнать больше о расширенных возможностях технологии.

Для развертывания операционных систем и приложений с помощью MDT среда должна соответствовать следующим требованиям к конфигурации программного обеспечения и компьютера.

Чтобы завершить работу с этим руководством, настройте компьютеры, перечисленные в следующей таблице. Эти компьютеры могут быть физическими компьютерами или виртуальными машинами с назначенными системными ресурсами.
