Управление образами клиентских ОС

Обзор форматов образа Windows

Существует три основных формата образа Windows.

- Windows image (.WIM)
- Virtual Hard Disk (.VHD/VHDX)
- Full Flash Update (.FFU)

Windows Imaging Format (WIM) — это файл-ориентированный формат образа диска. WIM файл может включать в себя несколько образов, которые определяются по числовому индексу или уникальному имени. Архитектура формата считается файл-ориентированной.

Virtual Hard Drive (VHD) — формат файла, содержащий полную структуру и содержание, сходные с жёстким диском. Используется для хранения виртуальных операционных систем, программ и других файлов в одном файле-образе, который можно открыть разными программами виртуализации или виртуальными машинами. Формат VHD используется в Virtual PC. Виртуальные жёсткие диски позволяют нескольким операционным системам одновременно находиться на одном компьютере. Это позволяет разработчикам тестировать программное обеспечение на различных операционных системах без приобретения дополнительного оборудования.

FFU - это формат системного образа, разработанный Microsoft. Файлы FFU используются для хранения системных образов Windows Phone и могут храниться на внешних устройствах хранения, таких как SD-карты. Файлы FFU обычно использовались для прошивки смартфонов Lumia с операционной системой Windows Phone.
Формат FFU редко используется, поскольку устройства Windows Phone больше не доступны на рынке. Старые устройства под управлением Windows Phone по-прежнему поддерживают файлы FFU, которые можно использовать для восстановления или обновления системы.
