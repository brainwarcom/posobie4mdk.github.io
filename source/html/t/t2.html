Обзор форматов образа Windows

Образ диска

Теперь несколько слов о самых распространенных форматах. В качестве универсального принят формат ISO (стандарт 9660). И именно он является самым распространенным. Но это абсолютно не значит, что нет других. Дело в том, что программное обеспечение для работы с образами от разных разработчиков сохраняет образы в собственных форматах

Практически все программы, так сказать, по негласной договоренности разработчиков, кроме собственного формата, умеют работать со всеми остальными. Исключение составляет только формат DMG, который в системах Windows не распознается. Нет, открыть его какой-нибудь программой можно, но какой в этом смысл, если содержимое будет нечитаемым и непригодным к использованию?

Файл каталога — это двоичный файл, который включает только параметры и пакеты в образе Windows. Файлы каталога (. clg) используется только Windows SIM и не используется другими средствами развертывания и не требуется для установки Windows. Когда Windows SIM-карта создает файл каталога, он запрашивает образ Windows для получения списка всех параметров и состояния каждого параметра в этом образе. Так как содержимое образа Windows может изменяться с течением времени, при обновлении образа Windows необходимо повторно создать файл каталога.
